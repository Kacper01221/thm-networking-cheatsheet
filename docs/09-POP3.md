# POP3 Protocol

POP3 (Post Office Protocol version 3) is a protocol used by email clients to retrieve emails from a mail server. It allows users to download their emails from the server to their local device.

## Port Numbers
- **Port 110**: Default port for unencrypted POP3 connections.
- **Port 995**: Default port for encrypted POP3 connections (POP3S).

## POP3 Commands
- **USER**: Used to specify the username for authentication.
- **PASS**: Used to specify the password for authentication.
- **STAT**: Returns the status of the mailbox, including the number of messages and total size.
- **LIST**: Lists the messages in the mailbox along with their sizes.
- **RETR**: Retrieves a specific message from the mailbox.
- **DELE**: Marks a specific message for deletion.
- **QUIT**: Ends the session with the mail server.

## Example Session
```
C: USER username
S: +OK
C: PASS password
S: +OK Logged in
C: STAT
S: +OK 2 300
C: LIST
S: +OK 2 messages:
1 100
2 200
C: RETR 1
S: +OK 100 octets
[Message content]
C: DELE 1
S: +OK message deleted
C: QUIT
S: +OK Goodbye
```

## Security Concerns
- **Eavesdropping**: POP3 can be vulnerable to eavesdropping when transmitting data over unsecured connections. It is recommended to use POP3S with SSL/TLS encryption.
- **Password Exposure**: Sending passwords in clear text can lead to exposure. Use secure authentication methods to mitigate this risk.
