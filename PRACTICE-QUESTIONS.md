# PRACTICE QUESTIONS

## OSI Model
- **Q1:** What layer of the OSI model is the TCP/IP model equivalent to?
  <details>
  <summary>Reveal Answer</summary>
  Transport layer
  </details>
  *Difficulty: Easy*

## IP
- **Q2:** What is the main purpose of IP addressing?
  <details>
  <summary>Reveal Answer</summary>
  To uniquely identify devices on a network.
  </details>
  *Difficulty: Easy*

## TCP/UDP
- **Q3:** Name a key difference between TCP and UDP.
  <details>
  <summary>Reveal Answer</summary>
  TCP is connection-oriented; UDP is connectionless.
  </details>
  *Difficulty: Medium*

## ICMP
- **Q4:** What does ICMP stand for and what is its primary use?
  <details>
  <summary>Reveal Answer</summary>
  Internet Control Message Protocol; used for diagnostic messages.
  </details>
  *Difficulty: Medium*

## ARP
- **Q5:** What does ARP do in a network?
  <details>
  <summary>Reveal Answer</summary>
  It maps IP addresses to MAC addresses.
  </details>
  *Difficulty: Easy*

## DNS
- **Q6:** How does DNS resolution work?
  <details>
  <summary>Reveal Answer</summary>
  It translates domain names to IP addresses.
  </details>
  *Difficulty: Medium*

## DHCP
- **Q7:** What is the purpose of DHCP in a network?
  <details>
  <summary>Reveal Answer</summary>
  Dynamic Host Configuration Protocol; automatically assigns IP addresses.
  </details>
  *Difficulty: Medium*

## HTTP/HTTPS
- **Q8:** What is the main difference between HTTP and HTTPS?
  <details>
  <summary>Reveal Answer</summary>
  HTTPS is HTTP over TLS/SSL; provides encryption.
  </details>
  *Difficulty: Medium*

## FTP
- **Q9:** What is FTP used for?
  <details>
  <summary>Reveal Answer</summary>
  File Transfer Protocol; used to transfer files.
  </details>
  *Difficulty: Easy*

## Linux Commands
- **Q10:** What command is used to list files in a directory?
  <details>
  <summary>Reveal Answer</summary>
  `ls`
  </details>
  *Difficulty: Easy*

## Advanced Topics
- **Q11:** Explain the function of a router in a network.
  <details>
  <summary>Reveal Answer</summary>
  Directs data packets between networks.
  </details>
  *Difficulty: Hard*

## Study Tips
- Review material regularly and practice hands-on labs.
- Use scenario-based questions to enhance understanding.
