This cheat sheet covers the essential networking protocols used in core networking tasks. It is designed for easy reference during networking studies and examinations.

Ta ściągawka zawiera podstawowe protokoły sieciowe używane w podstawowych zadaniach sieciowych. Została zaprojektowana jako łatwe odniesienie podczas nauki i egzaminów z zakresu sieci.

## Protocols Overview
### Application Layer / Warstwa aplikacji
- **HTTP**: HyperText Transfer Protocol. Used for transmitting web pages.
- **FTP**: File Transfer Protocol. Used for file transfers.
- **SMTP**: Simple Mail Transfer Protocol. Used for email sending.

### Transport Layer / Warstwa transportowa
- **TCP**: Transmission Control Protocol. Provides reliable, ordered, and error-checked delivery of a stream of data.
- **UDP**: User Datagram Protocol. Used for low-latency applications.

### Network Layer / Warstwa sieciowa
- **IP**: Internet Protocol. Responsible for addressing and routing packets.

### Data Link Layer / Warstwa łącza
- **Ethernet**: A common networking technology for local area networks.

### Physical Layer / Warstwa fizyczna
- **Cabling and connecters**: Hardware used for network connectivity.
